Penetration testing tools for the masses.

Blackhat Linux is a repository of penetration testing tools for RPM based linux distributions.

All tools are thoroughly tested before being added to the codebase to maintain the quality of the repository. This is a relatively new project and as we continue to grow the repository with the help of the community, we continue to make it better. If you find bugs, have suggestions or tool requests please feel free to visit our github repository or email us.
